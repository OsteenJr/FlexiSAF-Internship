Summary of Learning on JS and ES6 Array Methods

For the first week task, I studied some Javascript tutorials covering basics like variables, functionc, objects, loops, ES6 features and so many other subtopics. The key learning outcome of the week include:

  • Core JS Fundamentals such as variables, data types, operators, conditionals, loops, arrays, DOM Manipulation, events and async.
  • ES6-Specific Features such as Let/const for block scoping, arrow functions for concise syntax, destructuring for unpacking, spread operator for iteration/expansion, Sets/Maps for unique/ordered data, classes for OOP, and so many more.

Below is a demo and illustration of some of the things I learnt. It shows some array methods using a simple array: ["Arsenal", "Barcelona", "Chelsea", "Liverpool", "PSG", "Real Madrid", "Wolves"]. Click the buttons under each section to see each method in action.

1. Array.from(): Make Array from string

It turns a string like "Dortmund" into an array of characters.

2. Array.find():Find first match

Finds the first array element that matches the condition.

3. Array.findIndex():Index of First Match

Finds the index of the first array element matching a condition.

4. Array.entries():List Index/ Value Pairs

Shows each array element with its index.

5. Array.keys():List Indices

Shows the indices of the array.

6. Array.map():Transform an Array

Changes each array element.

7. Array.filter():Keep Matches

Keep array elements matching a condition.

8. Array.reduce():Combine to single value

Combines array into one value.

9. Array.forEach():Loop with some effects

Loops through array elements to perform an action.

10. Array.some():Check for any match

Checks if any array elements matches a condition.
